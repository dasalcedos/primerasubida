¡Bienvenido!

Para empezar presione una tecla cualquiera...

Es broma, se hace de la siguiente manera

Para ver los cursos ofertados:

	node registro.js 

Inscripción:

	node registro.js inscribir -i=1 -n=MiNombre -c:34422
	
donde:
-i ID del curso 1, 2 o 3
-n Nombre del estudiante
-c Cedula del estudiante

al realizar el registro la información queda guardada en un archivo denominado "curso.txt"

Muchas gracias!!
